# JLPT N5 단어 플래시카드

단어 목록을 하루 30개씩 학습하는 정적 웹앱입니다.

## 실행

```bash
python3 -m http.server 8765 --bind 127.0.0.1
```

브라우저에서 `http://127.0.0.1:8765/`를 엽니다.

## 기능

- 붙여넣은 단어 순서대로 하루 30개씩 학습
- 홈 화면에서 오늘의 30단어를 먼저 확인
- 히라가나를 주 표기로, 한자/표기를 보조 표기로 표시
- `플래시카드로 단어 학습하기` 버튼으로 별도 학습 페이지 이동
- 브라우저 내장 일본어 음성 합성으로 발음 듣기
- 셔플, 카드 이동 시 자동 발음, 발음 속도 조절
- 알고 있음 체크와 검색
